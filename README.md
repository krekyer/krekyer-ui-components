# Моя UI-библиотека

Это моя версия библиотеки компонентов на Vue 3.  
Сейчас в ней реализованы базовые UI-элементы, объединённые единым стилем с градиентной акцентной палитрой:

- **Кнопка (`UiButton`)** — три варианта оформления: `primary` (градиент), `secondary` (нейтральная), `outline` (без фона)
- **Поле ввода (`UiInput`)** — стилизованный инпут
- **Выпадающий список (`UiSelect`)** — стилизованный селект в том же дизайне
- **Группа полей (`UiField`)** — обёртка для подписей и инпутов

Все компоненты используют единую цветовую схему:
- Основной градиент: `#6e8efb → #a777e3`
- Вторичный фон: `#f3f4f6`
- Цвет текста: `#4b5563`
- Цвет границы: `#d1d5db`
